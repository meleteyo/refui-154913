⚠️
차순위 · V1 분석 알고리즘 책임 경계

"추천 투찰가"는 자사가 사전 계산? 파트너가 구현?

자사 사전 계산이면 본 견적으로 가능. 파트너 구현이면 일정 ±6~8주, 예산 ±50%.

▼
A. 자사 사전 계산 (가정)

자사 데이터팀이 분석 결과를 컬럼/뷰로 제공. 파트너는 read-only 시각화만.

✓ 본 견적 그대로 가능

B. 파트너 구현

분위수 회귀(LightGBM) + 임베딩 매칭. ML 인프라 + 학습 데이터 + 검증 필요.

✗ 별도 SOW (+5~8M, +6~8주)

⚙️
사전 준비 · V2+V3 자사 DB 연동 + PG 가맹

자사 DB 연동 방식 + PG 빌링 가맹 진행 상태

REST API vs DB 직결 결정. PG 빌링은 통상 2~4주 심사 → W1 전 신청 필수.

▼
  • ▸ REST API (권장) — 결합도 낮음, 향후 자사 시스템 변경에도 우리 코드 영향 최소
  • ▸ DB 직결 — 결합도 높음, 보안 협의 필요
  • ▸ 일배치 덤프 — 실시간성 약함, MVP 시나리오에 적합
  • ▸ 토스페이먼츠 빌링 가맹은 통상 2~4주. 신규 사업이면 +2주.
