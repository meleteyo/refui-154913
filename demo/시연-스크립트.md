## 녹화 가이드 (macOS)

1. 프로토타입을 1280×720 창 크기로 띄우기 (브라우저 zoom 100%)
2. ⌘ Cmd+Shift+5 → "선택 부분 기록" → 프로토타입 영역 드래그
3. "녹화" 버튼 클릭 후 ▶ 재생 클릭, 위 스크립트 따라 멘트
4. 녹화 중지 후 mov 파일을 ffmpeg로 mp4 변환 (선택):
   ```
   ffmpeg -i demo.mov -vcodec h264 -acodec aac demo.mp4
   ```
5. Loom 무료 플랜 대신 자체 mp4 사용 가능
