자주 묻는 질문
1개월 무상 유지보수 후엔 어떻게 되나요? ▾
무상 유지보수 기간(1개월) 종료 후에는 유상 유지보수(월정액 또는 건별 견적) 계약으로 전환 가능합니다. 인수인계 문서를 완성하므로, 별도 파트너 없이 자체 운영도 가능합니다.
코드 소유권은 누구에게 있나요? ▾
최종 납품 이후 모든 소스코드·산출물의 소유권은 발주자에게 이전됩니다. 위시켓 표준 계약 기준으로 진행하며, 별도 제한 없습니다.
다른 개발자도 코드를 받을 수 있나요? ▾
네. 한국에서 가장 많이 쓰는 표준 기술(Java/Spring Boot + PostgreSQL + Next.js)로 개발합니다. 기능별 모듈 분리 구조로 설계하며, 인수인계 문서(사용 설명서 + API 명세 + 데이터 구조 + 배포 가이드)를 포함하므로 다른 개발자도 인수 가능합니다.
인수인계 품질은 어떻게 보장되나요? ▾
22주차에 산출물 패키지를 제공합니다: 소스코드 + 사용 설명서 + 데이터 구조도 + 배포 스크립트 + 운영 가이드 + 1개월 무상 유지보수. 인수인계 항목을 미팅에서 미리 합의합니다.
기술 종속성 우려는 없나요? ▾
한국에서 가장 많이 쓰는 표준 기술을 사용합니다. 특수 프레임워크나 독점 서비스를 사용하지 않아 향후 다른 파트너도 유지보수 가능합니다. 특정 개발자에게 종속되지 않는 구조입니다.
개발 중 변경이 가능한가요? ▾
네. 1주 단위 동작 데모 사이클로 진행하므로 매주 피드백을 반영합니다. 큰 변경은 미팅에서 일정·예산을 재합의하며, 작은 개선은 다음 주 데모에 반영합니다. 납품 직전 급변경만 피하시면 됩니다.
